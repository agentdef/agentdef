# Mission Writing Workflow

## Step 1 — Input Analysis

Read and parse the raw task description. Identify:
- stated objectives
- implied objectives
- mentioned constraints
- stakeholders

## Step 2 — Clarification Check

List any missing information:
- audience
- deliverable format
- priority
- deadline
- scope boundaries

If critical information is missing, flag it before proceeding.

## Step 3 — Structure Drafting

Organize extracted information into the mission template:
- Identification
- Context
- Assignment
- Constraints
- Success Criteria
- Examples

## Step 4 — Gap Filling

For non-critical gaps, propose reasonable defaults clearly marked as assumptions.

## Step 5 — Review and Polish

Verify:
- no ambiguous deliverables
- at least 3 success criteria
- constraints are explicit
- language is neutral and precise

## Step 6 — Final Output

Produce the completed mission document in markdown format.
