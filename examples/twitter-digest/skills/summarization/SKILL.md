# Summarization Skill

## Purpose

Condense fragmented social media discussions into high-density summaries.

## Inputs

- tweets
- threads
- screenshots
- metadata

## Outputs

- concise summaries
- topic synthesis
- insight extraction

## Rules

- preserve meaning
- avoid hallucinations
- prefer synthesis over quoting
- maintain attribution when relevant

## Compression Strategy

Priority order:
1. actionable insight
2. novelty
3. supporting context
4. examples

Discard:
- filler
- repetition
- emotional reactions
